## 使用说明
#### SearchPwdFromShadow.py
```
python SearchPwdFromShadow.py -d <shadow dir> -p <pwd file> -o <out file>
```

#### shadowFilter.py 
```
python shadowFilter.py -d <target dir> -c <column> -o <directory>
```
|参数|描述|
|:---:|:---:|
|-F|shadow文件|
|-d|目录|
|-f|过滤用户|
|-m|模式|
|-c|列|
|-o|输出|
|--iuser|包含用户|
|--euser|排除用户|
